Swiggy Food Delivery Analysis Dashboard | Excel

This project analyzes Swiggy food delivery data containing over 197,000 menu items and restaurant records across multiple cities and states in India. Using Excel, the dataset was cleaned, transformed, and visualized to uncover insights into restaurant performance, food categories, pricing trends, customer ratings, and order patterns.

The dashboard provides interactive analysis of revenue, average order value (AOV), ratings, rating counts, popular food categories, and restaurant performance across different locations. It helps identify customer preferences, top-performing restaurants, pricing behavior, and regional food trends to support business decision-making.

Key Insights
Analyzed ₹53M+ worth of menu pricing data.
Evaluated restaurant performance using ratings and customer review counts.
Identified top food categories and popular dishes across cities.
Compared pricing trends across restaurants and locations.
Monitored quarterly and weekly ordering patterns.
Built an interactive Excel dashboard using Pivot Tables, Charts, Slicers, and KPIs.
